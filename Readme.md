`# Pour ex`ectuer le programme:
- Etre sous Linux (ou wsl)
- run les commandes suivantes :
  ```Bash
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  ```
- simplement run `env.py`

Si vous souhaitez modifier le nombre d'individus au lancement de la simulation, il suffit de modifier la liste l.180 dans `env.py` 

## Pour stopper le programme:
Il suffit d'appuyer sur le bouton quitter de l'affichage
